# golang-dev-lab
Documenting my journey through Golang: forging scripts, deep-diving into the standard library, and building scalable prototypes.

## 🚀 Overview
Repositori ini berfungsi sebagai laboratorium pengembangan untuk mengeksplorasi ekosistem **Go (Golang)**. Fokus utama mencakup penguasaan sintaksis, pemanfaatan *standard library*, hingga implementasi kontainerisasi menggunakan **Docker**.

## 🛠️ Tech Stack & Environment
- **Language:** Go (Golang)
- **OS:** Ubuntu 22.04 LTS
- **Tools:** Git
- **IDE:** VS Code
